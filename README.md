# Yvee Noir Website

## Project info

This repository contains the Yvee Noir website built with Vite, React, and TypeScript.

## How to run locally

Requirements: Node.js and npm.

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Building for production

```sh
npm run build
npm run preview
```

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

If you want assistance deploying or connecting a custom domain, I can help set that up.
